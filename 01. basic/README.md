## Datatypes

In this section, you will tackle problems related to data types. To tackle these challenges, leverage the native functions provided by each programming language to address these issues.

### Strings

1. **Reverse a String:**
    - *Description:* Given a string, reverse the order of characters.
    - *Example Input:* `"hello"`
    - *Example Output:* `"olleh"`

2. **Count the Characters:**
    - *Description:* Count and return the number of characters in a given string.
    - *Example Input:* `"programming"`
    - *Example Output:* `11`

3. **Uppercase and Lowercase:**
    - *Description:* Convert a string to both uppercase and lowercase.
    - *Example Input:* `"Code"`
    - *Example Output:* `"CODE"`, `"code"`

4. **Concatenate Strings:**
    - *Description:* Concatenate two strings and return the result.
    - *Example Input:* `"Hello"`, `"World"`
    - *Example Output:* `"HelloWorld"`

5. **Substring Extraction:**
    - *Description:* Extract a substring from a given string.
    - *Example Input:* `"programming"`, start: `3`, length: `5`
    - *Example Output:* `"gramm"`

6. **String Length:**
    - *Description:* Determine and return the length of a given string.
    - *Example Input:* `"exercise"`
    - *Example Output:* `8`

7. **Check Palindrome:**
    - *Description:* Check if a given string is a palindrome.
    - *Example Input:* `"level"`
    - *Example Output:* `true`

8. **Replace Characters:**
    - *Description:* Replace a specific character in a string with another character.
    - *Example Input:* `"apple"`, replace `'p'` with `'m'`
    - *Example Output:* `"amole"`

9. **Count Vowels:**
    - *Description:* Count and return the number of vowels in a given string.
    - *Example Input:* `"programming"`
    - *Example Output:* `3`

10. **Find Substring:**
    - *Description:* Determine if a substring exists in a given string.
    - *Example Input:* `"Hello World"`, substring: `"World"`
    - *Example Output:* `true`


### Numbers

1. **Addition:**
   - *Description:* Given two numbers, calculate their sum.
   - *Example Input:* `5, 7`
   - *Example Output:* `12`

2. **Subtraction:**
   - *Description:* Given two numbers, calculate their difference.
   - *Example Input:* `10, 4`
   - *Example Output:* `6`

3. **Multiplication:**
   - *Description:* Given two numbers, calculate their product.
   - *Example Input:* `3, 8`
   - *Example Output:* `24`

4. **Division:**
   - *Description:* Given two numbers, calculate their quotient.
   - *Example Input:* `15, 3`
   - *Example Output:* `5`

5. **Square of a Number:**
   - *Description:* Calculate the square of a given number.
   - *Example Input:* `9`
   - *Example Output:* `81`

6. **Cube of a Number:**
   - *Description:* Calculate the cube of a given number.
   - *Example Input:* `4`
   - *Example Output:* `64`

7. **Check Even or Odd:**
   - *Description:* Determine if a given number is even or odd.
   - *Example Input:* `14`
   - *Example Output:* `Even`

8. **Find Maximum:**
   - *Description:* Given two numbers, find and return the larger of the two.
   - *Example Input:* `22, 18`
   - *Example Output:* `22`

9. **Calculate Average:**
   - *Description:* Given three numbers, calculate and return their average.
   - *Example Input:* `15, 20, 25`
   - *Example Output:* `20`

10. **Check Prime Number:**
    - *Description:* Determine if a given number is a prime number.
    - *Example Input:* `7`
    - *Example Output:* `Prime`

## Booleans

1. **Logical AND Operation:**
    - *Description:* Implement a function that performs a logical AND operation on two boolean values.
    - *Example Input:* `true, false`
    - *Example Output:* `false`

2. **Logical OR Operation:**
    - *Description:* Implement a function that performs a logical OR operation on two boolean values.
    - *Example Input:* `true, false`
    - *Example Output:* `true`

3. **Logical NOT Operation:**
    - *Description:* Implement a function that performs a logical NOT operation on a boolean value.
    - *Example Input:* `true`
    - *Example Output:* `false`

4. **Check Equality:**
    - *Description:* Implement a function that checks the equality of two boolean values.
    - *Example Input:* `true, true`
    - *Example Output:* `true`

5. **Check Inequality:**
    - *Description:* Implement a function that checks the inequality of two boolean values.
    - *Example Input:* `true, false`
    - *Example Output:* `true`

6. **Conditional Operator:**
    - *Description:* Use the conditional operator (`? :`) to return different values based on a boolean condition.
    - *Example Input:* `true`
    - *Example Output:* `It's true!`

7. **Toggle Boolean:**
    - *Description:* Implement a function that toggles the value of a boolean variable.
    - *Example Input:* `true`
    - *Example Output:* `false`

8. **Check If All are True:**
    - *Description:* Implement a function that checks if all boolean values in a given list are true.
    - *Example Input:* `[true, true, false]`
    - *Example Output:* `false`

9. **Check If Any is True:**
    - *Description:* Implement a function that checks if at least one boolean value in a given list is true.
    - *Example Input:* `[false, false, true]`
    - *Example Output:* `true`

10. **Boolean to Number Conversion:**
    - *Description:* Implement a function that converts a boolean value to a corresponding numeric value (0 for false, 1 for true).
    - *Example Input:* `true`
    - *Example Output:* `1`
